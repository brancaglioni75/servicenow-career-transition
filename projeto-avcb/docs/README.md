
# 📌 Regras de Negócio - Projeto AVCB

## Objetivo
Controlar vencimento e renovação do AVCB de unidades hospitalares veterinárias.

## Regras Implementadas

### 1. Cadastro obrigatório
Campos obrigatórios:
- Unidade
- Responsável
- Data de Emissão
- Data de Vencimento
- Status

### 2. Controle de Status
Valores possíveis:
- Vigente
- A vencer
- Em renovação
- Vencido
- Renovado

### 3. Responsabilidade
Cada unidade possui um responsável definido.

### 4. Histórico
Todos os registros ficam armazenados para consulta futura.
